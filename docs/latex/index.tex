\hypertarget{index_intro_sec}{}\doxysection{Introduction}\label{index_intro_sec}
The note-\/python library for communicating with the \href{https://blues.io}{\texttt{ Blues Wireless}} \mbox{\hyperlink{classnotecard_1_1notecard_1_1_notecard}{Notecard}} via serial or I2C.

This library allows you to control a \mbox{\hyperlink{classnotecard_1_1notecard_1_1_notecard}{Notecard}} by writing a Python, Circuit\+Python, or Micro\+Python program. You may programmatically configure \mbox{\hyperlink{classnotecard_1_1notecard_1_1_notecard}{Notecard}} and send Notes to \href{https://notehub.io}{\texttt{ Notehub.\+io}}.\hypertarget{index_dependencies}{}\doxysection{Dependencies}\label{index_dependencies}
When used in a Python program, this library depends on the \href{https://pypi.org/project/python-periphery/}{\texttt{ periphery}} and \href{https://pypi.org/project/filelock/}{\texttt{ filelock}} modules.

In addition, this library requires a physical connection to a \mbox{\hyperlink{classnotecard_1_1notecard_1_1_notecard}{Notecard}} over I2C or Serial to be functional.\hypertarget{index_author}{}\doxysection{Author}\label{index_author}
